# DSP_FastDrones
This mod increases the speed of Mecha Drones to 299 (previous max speed was 59). It also decreases energy usage of drones by a set factor (default: 50.0). 
The mod does not require a new game start, it modifies the speed on the fly, and will not affect your savefile, so you can go back to old speeds if you want.  

Many thanks to @Appun for figuring out how to break the speed limit and for suggestions/code.
